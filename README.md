# Менеджер задач — Практическое занятие 13

React-приложение для управления списком задач, созданное с использованием Vite.

## Функциональность

- Добавление новых задач
- Отметка задач как выполненных
- Удаление задач
- Редактирование задачи по двойному клику
- Фильтрация задач: все / активные / выполненные
- Счётчик оставшихся задач
- Переключение светлой и тёмной темы
- Сохранение задач и темы в `localStorage`

## Стек

- [React](https://react.dev/) — библиотека для построения интерфейсов
- [Vite](https://vitejs.dev/) — инструмент сборки и сервер разработки

## Структура проекта

```
src/
├── main.jsx                  # Точка входа
├── App.jsx                   # Корневой компонент
└── components/
    ├── TodoItem.jsx          # Компонент одной задачи
    ├── AddTodoForm.jsx       # Форма добавления задачи
    └── TodoFilters.jsx       # Фильтры и счётчик
```

## Запуск проекта

```bash
npm install
npm run dev
```
